Veldrid 4.4.0 Beta 2

July 25, 2018

Veldrid version 4.4.0 beta 2 is now available. This includes a few minor bug fixes before 4.4.0 becomes stable.

All Changes

  • Fix a couple new Shader-related problems [3bcfb2c]
  • [Metal] Fix disposal of MTLPipeline with no specialized functions. [a632368]
  • [Vulkan] Allow VkCommandList to be disposed immediately if there's no pending submissions. [85fcf6d]
  • [Vulkan] Avoid hidden enumerator allocations. [c748cdc] (@SomeAnon42)
  • Use CallingConvention.StdCall for all imported OpenGL functions. [5d8abd6] #100
  • [Vulkan] Ensure FixedUtf8String is null-terminated. [2fdeda3]
  • [Vulkan] Fix several validation errors in the new SDK. [5f44a73]
  • Bump ImGui.NET dependency to 0.4.7. [28a9d74]
  • [OpenGL] Use Unsafe.ReadUnaligned in OpenGLNoAllocCommandEntryList. [c771b33]
  • Make Sdl2Native.LoadFunction public. [1a398fa] #97
  • [D3D11] Don't fail Shader compilation unless result indicates failure. [2164433]
  • Ensure that Swapchain Textures have correct properties. [05a4171]
  • Bump ImageSharp dependency to beta0004. [f6196c3]
  • Avoid allocations when calling GetHashCode or Equals on certain structs [86f5ac8] (@SomeAnon42)
  • [SDL2] Add Sdl2Window.MouseDelta [91f5428]
  • Use CommandList.UpdateBuffer in ImGuiRenderer. [6d3a66b] #73
  • Improve this Doc
