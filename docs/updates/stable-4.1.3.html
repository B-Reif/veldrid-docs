Veldrid 4.1.3

Veldrid version 4.1.3 is available. This is a small patch with some minor fixes and improvements.

All Changes

Veldrid

  • [Vulkan] Use correct Framebuffer dimensions when setting Swapchain.SyncToVerticalBlank. [1161758]
  • [Vulkan] Fix an issue with the tracking of clear values in VkCommandList. [c337c56]
  • [Vulkan] Fix an erroneous comparison in VkTexture.cs. [e84db0e]
  • Defensively copy some arrays in the creation of resources, when necessary. [0e0fa87]
  • [Vulkan] Add more needed synchronization to VkGraphicsDevice. [8d5490a]
  • [D3D11] Synchronize DXGI usage with the immediate context (per MSDN). [de32772]
  • [Metal] Support CommandList.CopyBuffer and UpdateBuffer when region is unaligned. [903ea1c]
  • [Metal] Fix condition for compute buffer copy path. [10dcb69]
  • [Vulkan] Fix aspect mask for depth-stencil clears. [5f6ceb9]
  • [OpenGL] Fix issues with texture binding and state tracking. [4e25323]
  • [Vulkan] Fix an issue with how render passes are specified. [88c8a87]
  • [Vulkan] Fix multiple vertex bindings in Vulkan. [271a9bb]
